## 🎯 What is Cluster Heartbeat?

Cluster Heartbeat is an AI-powered system that continuously monitors GPU clusters using Prometheus and NVIDIA DCGM metrics, producing a unified workload fingerprint that powers three intelligent services:

1. **Predictive Failure Detection** - Catch GPU failures hours before they happen
2. **Smart GPU Scheduling** - Schedule workloads by actual behavior, not static requests  
3. **GPU Cost Optimization** - Identify and eliminate idle GPU spend automatically
